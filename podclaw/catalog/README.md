## Modelo de Precios

**Fórmula:**
```
Precio Venta (EUR) = (Coste Base + Envío EU) / (1 - Margen Objetivo)
```

**Márgenes objetivo por categoría:**
- Apparel (camisetas, sudaderas): **40-45%**
- Drinkware (tazas, botellas): **45-55%**
- Home & Living (velas, canvas): **50-60%**
- Accesorios (tote bags, llaveros): **55-65%**
- Bordados premium: **45-55%**

**Notas importantes:**
- Precios base con **Printify Premium** (descuento ~20%)
- Envío intra-EU desde proveedores europeos
- IVA no incluido en costes base (se añade según país del cliente)
- Tipo de cambio USD/EUR = 0.92 (referencia Feb 2026)
