## Fórmula de Pricing

```
Coste Total = Coste Base (Premium) + Envío EU (1er artículo) + Branding
Precio Venta = Coste Total / (1 - Margen Objetivo)
Beneficio Bruto = Precio Venta - Coste Total
```

**Ejemplo con branding:** Camiseta Bella+Canvas 3001
- Coste base Premium: €8.08
- Envío EU: €3.76
- Neck label: €0.40
- Total: €12.24
- Margen objetivo: 46%
- PVP = €12.24 / (1 - 0.46) = **€22.67 → €22.95**
- Beneficio bruto: €10.71 por unidad

**Ejemplo sin branding:** Camiseta Bella+Canvas 3001
- Coste base Premium: €8.08
- Envío EU: €3.76
- Total: €11.84
- Margen objetivo: 46%
- PVP = €11.84 / (1 - 0.46) = **€21.93 → €21.95**
- Beneficio bruto: €10.11 por unidad

## Break-Even Analysis

Con costes fijos de ~€24/mes y margen promedio de ~€15/ud:
- **Break-even:** ~2 ventas/mes
- **Objetivo realista mes 1-3:** 10-20 ventas/mes = €150-300 beneficio
- **Objetivo mes 6+:** 50-100 ventas/mes = €750-1,500 beneficio
- **Objetivo año 1:** 100-200 ventas/mes = €1,500-3,000 beneficio
